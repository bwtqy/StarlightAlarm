# 星星闹钟助手

**核心逻辑：被「星星充电」App 链式启动 → 触发系统 7:58 闹钟**

## 工作原理

```
星星充电 App 启动
    ↓
MIUI 链式启动管理触发
    ↓
本 App 被自动拉起（无需用户手动）
    ↓
MainActivity.onCreate 执行
    ↓
判断当前时间是否在 0:00~7:59 之间
    ├── 否 → 显示提示，不设闹钟
    └── 是 → 跳转系统闹钟设置（预填 7:58）
                  ↓
              用户点『确定』→ 闹钟生效
```

## 配置步骤

### 1. 编译安装
Android Studio → 导入 `StarlightAlarm` → Run 到手机

### 2. 配置 MIUI 链式启动（关键）
```
设置 → 应用设置 → 应用管理 → 星星闹钟助手
    → 链式启动管理 → 允许
    → 自启动 → 允许
    → 省电策略 → 无限制
```

### 3. 配置触发源
```
设置 → 应用设置 → 应用管理 → 星星充电
    → 链式启动管理 → 添加「星星闹钟助手」
```

## 与之前方案的区别

| 项 | 旧方案（错误） | 新方案（正确） |
|---|---|---|
| 监控方式 | 后台Service + UsageStats | 依赖MIUI链式启动 |
| 常驻后台 | 需要 | **不需要** |
| 需要的权限 | PACKAGE_USAGE_STATS | 仅 SET_ALARM |
| 代码量 | 较复杂 | 极简 |
| 可靠性 | 依赖服务保活 | 依赖系统拉起 |

## 注意事项

- 必须使用 MIUI 12.5+ 系统（带链式启动管理）
- 首次需手动配置一次链式启动关系
- 闹钟设置会弹出系统确认界面，需用户点『确定』
